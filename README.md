# 教師資格檢定



### 國民小學師資類評量指標

| 應試科目                 | 包含科目（大約）                                             | 評量指標                                                     |
| ------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| (一)國語文能力測驗       | 國語文                                                       | **1.語文理解：**包括語文知識與閱讀理解。<br /> (1)語文知識： A.字詞理解與運用分析。 B.文化常識與應用文書。 <br />(2)閱讀理解： A.內容意旨。 B.篇章結構與風格欣賞。<br />**2.溝通表達：**能以通順語句、適當結構，抒發主觀經驗並整合客觀資訊，以達成有效溝通。 |
| (二)數學能力測驗         | 數學 <br /> 數學教材教法                                     | **1.普通數學：** <br />(1)理解數學概念之意義及概念間連結。<br /> (2)運用合理方法與步驟執行數學程序。<br /> (3)應用數學知識來解決數學或生活中問題。<br />**2.數學教材教學：**<br /> (1)理解國民小學數學課程內容及教材脈絡，並應用於數學教學。<br /> (2)理解國民小學學童數學概念之發展與迷思，並應用於數學教學。<br /> (3)理解國民小學數學教學策略與多元評量方法，並應用於數學教學。 |
| (三)教育理念與實務       | 教育哲學<br /> 教育社會學<br /> 教育行政                     | (1)了解主要教育思潮與理論之意義、規準、本質、目的及功能，並應用於各級學校教育。 <br />(2)了解並應用教育內外在社會環境與社會脈絡，包括巨觀、微觀二層面，如教育與社會公平、社會變遷、社會關係等。<br /> (3)了解我國主要教育行政與教育制度、教育法規與教育政策、學校實務及教育改革趨勢，並應用於國民小學教育情境。 <br />(4)了解教師專業倫理與實踐之內涵，並應用於國民小學。 |
| (四)學習者發展與適性輔導 | 兒童發展與輔導<br /> 輔導原理<br /> 教育心理學<br /> 班級經營<br /> 特殊教育概論 | (1)了解兒童身心發展理論、不同社經及文化背景之發展差異，並應用於教學與輔導。<br /> (2)了解主要學習理論(包括學習策略)與動機理論，並應用於國民小學學生學習與輔導。<br /> (3)辨識特殊需求學生之身心特質及鑑定歷程，並提供適切支持與轉介。<br /> (4)了解班級經營、正向支持的原理、方法與技巧，並應用於培養學生自律與自治，促進親師生關係及營造友善學習環境。 <br />(5)了解主要輔導理論、方法與技巧、輔導機制與資源、輔導倫理與主要法規，並應用於協助學生適應與發展。 |
| (五)課程教學與評量       | 教學原理<br /> 課程發展與設計<br /> 學習評量                 | 本指標中所指稱之「了解」與「應用」，涵蓋師資生之情境應用、高層次思考、反思修正等表現，包括下列五項： <br />(1)了解課程主要原理、課程發展與設計、課程評鑑之原則，以應用於國民小學課程、教學及評量。<br /> (2)考量社會變遷中之重要議題、課程政策與改革趨勢，以發展課程、教學及評量。<br /> (3)了解教學之主要理論與模式，以設計、實施、改善國民小學教學。<br /> (4)了解多元教學方法與策略、學習科技與資源，以應用於國民小學教學規劃與實踐(包括探究與實作)。<br /> (5)了解與使用多元評量方法(包括使用科技)，以檢視學習進展，促進學生自我成長，及指引與調整教學。 |
